Disaster Response System

Using Omnigraffle, I wireframed a potential mobile app for a disaster response system. In the existing system, hazardous material sensors transfer gathered data to a station on the same wifi network. That local system then converts the proprietary sensor data to the Common Alerting Protocol (CAP) format. In CAP form, the alerts are distributed to federal, civil, and private emergency operations agencies. The mobile app would allow users to view sensor live readings, read sensor alert information, and send out simulated sensor alarms.

365 Daily Design Project

As with 365 daily photograph projects, I tried to write and design a page layout each day. Writing or designing something every day is hard enough alone, so trying to do both was a little too lofty a goal. The point of these projects is to establish consistency and quantity over quality---eventually quality improves because you're getting so many reps in. And even though I only got through about a month, I did create a few pages that I really like.

Final Fantasy V Thoughts

For this day, I wrote about the greatest accomplishment of my life: beating a Japanese copy of Final Fantasy V in elementary school. It was still a few years away from an English release so I had no access to English guides. The only way to advance the story was through brute exploration. I'm guessing there were guides online, but the internet basically didn't exist to me then. Other things that didn't quite exist: my social life, girls.
